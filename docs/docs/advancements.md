这些是本服独有的进度，每个完成进度都会奖励一些:star:（目前暂时没用，请保存好）和独特的物品。本文仅列举所有的进度信息，不展开说明奖品信息，这是未来工作的一部分。

- :star: : 1
- 奖品: 新手物资

- :star: : 5
